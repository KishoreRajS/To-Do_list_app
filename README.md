# To-Do List CLI Application

# Project Overview

The To-Do List CLI Application is a simple command-line task management system developed using Python. It allows users to add, view, and remove tasks while storing them permanently in a text file.
The application demonstrates the implementation of file handling, lists, string manipulation, functions, loops, and conditional statements.

# Features
- View all tasks
- Add new tasks
- Remove completed tasks
- Save tasks permanently in a text file
- Automatically loads saved tasks when the application starts
- Menu-driven command-line interface

#Technologies Used
- Python 3.x
- VS Code / Any Python IDE
- Command Line Interface (CLI)
